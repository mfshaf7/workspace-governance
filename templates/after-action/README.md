# After-Action Template

Use `reviews/after-action/TEMPLATE.yaml` for the review record shape.

If the signal is real but the durable control shape is still unclear, start
with `templates/improvement-candidate/` instead of forcing a premature
after-action.

Use `scripts/record_after_action.py` when you want a scaffolded record created
with one initial lesson and the obvious metadata filled in from the command
line.
