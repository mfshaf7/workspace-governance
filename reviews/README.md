# Reviews

This directory contains governed review outputs owned by `workspace-governance`.

Current review class:

- `delegation-journal/`
  - governed audit records for delegated execution, packet scope, and
    integration outcome
- `improvement-candidates/`
  - fast triage records for likely repeated misses or late discoveries before
    they become a full after-action review
- `after-action/`
  - lessons learned, closure state, and linked durable controls for important
    misses, late discoveries, or repeated operator pain
